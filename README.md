### * 주요 기능 안내 *

한 달 단위로 예산을 정해놓고 하루에 얼마나 사용했는지 
돈을 기록하며 소비습관을 형성시켜주는 앱입니다.

돈을 과도하게 사용하지 못하게 내가 오늘 얼마를 사용할 수 있는지 계산해서 알려줍니다.
